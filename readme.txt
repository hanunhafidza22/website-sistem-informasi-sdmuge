Cara Menjalankan Website SD Muhammadiyah Bojonggede Secara Lokal :

1. Install software XAMPP di komputer.
2. Akses phpMyAdmin pada web browser (localhost/phpmyadmin).
3. Buat database baru dengan nama disesuaikan dengan keinginan dan kebutuhan, lalu import database pada bagian "Import" dan pilih tipe data yaitu sql (db_sekolah untuk database sekolah dan elearning untuk database e-learning).
4. Extract zip/rar menjadi folder website dan e-learning (masing-masing/terpisah) dan copy ke folder -> "xampp/htdocs" yang sebelumnya telah diinstal.
5. Akses website sistem informasi melalui -> localhost/nama_folder_website.
6. Akses website dashboard admin melalui -> localhost/nama_folder_website/admin.
7. Akses E-learning melalui -> localhost/nama_folder_elearning.
=================================================================================

Cara Edit Kode Program (apabila diperlukan oleh Admin):
1. Buka file kode program menggunakan text editor seperti Notepad, Notepad++, Visual Studio Code (recommend).
2. Edit baris pada file kode program.
3. Simpan perubahan kode program.
4. Test program dengan akses secara lokal.

================================================================================
Cara Penggunaan Bot Dialogflow.
1. Akses website Google Dialogflow.
2. Buat akun/login ke Google Dialogflow.
3. Buat Agent (nama bot) dan atur ke bahasa Indonesia (atau bahasa lainnya).
4. Import data bot dari zip/rar yang tersedia.
5. Edit data pengetahuan bot pada masing-masing intent.
6. Simpan perubahan.
7. Test pada bot.
